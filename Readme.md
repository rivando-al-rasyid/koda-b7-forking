# WELCOME TO FORKING

## HOW TO COLLABORATE
1. Fork from source repo to your own repo
2. Clone your own repo to your local directory
3. Add new remote (ex. upstream) which contains source repo url
4. Create your own feature branch for updating
5. Add and Commit your changes
6. Push local repository to your own repo
7. Create new Pull Request (PR) to source repo
8. Wait for owner to approve your PR
